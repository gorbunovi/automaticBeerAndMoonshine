Система для управления пивоварней, дистиляцией/ректификацией и автоклавом.

Данная система реализованна на микропроцессоре STM32F407VE. Взаимодействие с пользователем осуществляется через экран с тачскином.
Также управление меню продублированно кнопками.

Меню разбито на три блока.
1. Пивоварня.
2. Дистиляция - Ректификация.
3. Автоклавирование.

Пивоварня имеет два блока
1. Затирание.
2. Варка.

Дистиляция - Ректификация имеет 2 блока
1. Перегонка.
2. Обработка продукта.


Основные функции
Удержание заданной температуры посредством ПИД регуляции с регулированием мощьности тэна
